Reference

CLI Reference

Automate BC2 Fabric workflows directly from your terminal using the official CLI.

Commands

Command Description
bc2 init Scaffold a new workspace with boilerplate configuration.
bc2 deploy Deploy the current workspace to the configured environment.
bc2 status Inspect the health of recent deployments and connectors.
bc2 logs Stream structured logs filtered by connector or route.

Configuration file

{
  "workspace": "my-app",
  "tenantId": "acme-production",
  "environment": "staging",
  "connectors": ["salesforce", "s3-archive"]
}

Scripting

#!/usr/bin/env bash
set -euo pipefail

bc2 init --workspace nightly-sync
bc2 deploy --workspace nightly-sync --env staging
bc2 logs --workspace nightly-sync --since 1h
