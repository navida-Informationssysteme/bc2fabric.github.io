Webhooks

Receive event notifications in real time as BC2 Fabric processes and delivers data.

Event types

  • delivery.succeeded — emitted when a message is acknowledged.
  • delivery.failed — triggered after all retry attempts are exhausted.
  • delivery.retried — sent when a retry is scheduled.

Endpoint requirements

  • Must respond with a 2xx status within 10 seconds.
  • Support HTTPS with TLS 1.2+.
  • Verify the X-BC2-Signature header for authenticity.

Signature verification

const crypto = require("crypto");

function verifySignature(payload, signature, secret) {
  const hmac = crypto
    .createHmac("sha256", secret)
    .update(payload, "utf8")
    .digest("hex");

  return crypto.timingSafeEqual(Buffer.from(hmac), Buffer.from(signature));
}
