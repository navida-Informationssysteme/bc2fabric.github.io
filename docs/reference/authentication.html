Authentication

Authenticate API requests with OAuth 2.0 client credentials or signed service tokens.

OAuth 2.0 client credentials

POST https://auth.bc2fabric.com/oauth/token
Content-Type: application/x-www-form-urlencoded

grant_type=client_credentials&client_id=...&client_secret=...

Responses include an access token that expires in 60 minutes. Refresh tokens are not issued for this flow.

Service tokens

Service tokens are JWTs signed with your organization's managed keys. Generate them using the CLI: 

bc2 tokens create --scope "events:write" --ttl 15m

SDK configuration

import { Client } from "@bc2fabric/sdk";

const client = new Client({
  clientId: process.env.BC2_CLIENT_ID,
  clientSecret: process.env.BC2_CLIENT_SECRET,
});
