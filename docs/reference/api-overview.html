API Overview

The BC2 Fabric API is a RESTful interface with predictable resources and consistent error handling patterns.

Base URL

https://api.bc2fabric.com/v1

Common headers

Authorization: Bearer <token>
X-BC2-Tenant: <tenant-id>
Content-Type: application/json
Accept: application/json

Pagination

All list endpoints support cursor-based pagination via the next_cursor field. Pass the cursor value in the cursor query parameter to fetch the next page.

Error format

{
  "error": {
    "code": "resource_not_found",
    "message": "The requested resource does not exist.",
    "request_id": "b9e8f8"
  }
}
