BC2Fabric Prerequisites

Review the infrastructure, access, and configuration requirements before installing the BC2Fabric workload in your Microsoft tenant.

Download the full installation guide (PDF)

1. Business Central environment readiness

  • Confirm you are running Dynamics 365 Business Central online (version 26 or later) in the production or sandbox environment where BC2Fabric will synchronize data.
  • Install the BC2Fabric extension from Microsoft AppSource and assign it to the companies that will publish telemetry and application data to Fabric.
  • Create a dedicated Business Central user (service account) with the BC2Fabric user group or equivalent permissions to read the target tables and system events.
  • Verify that the environment time zone, job queue frequency, and data retention meet the synchronization requirements outlined in the installation checklist.

2. Microsoft Entra ID application registration

  • Register a single-tenant application in Microsoft Entra ID to represent BC2Fabric and capture the application (client) ID and directory (tenant) ID.
  • Create a client secret that will be shared with the BC2Fabric workload and note its expiration date for future rotations.
  • Grant the application delegated permissions for Dynamics 365 Business Central, Azure Key Vault, and any auxiliary APIs required by your deployment. Admin consent is needed before installation.
  • Add the service account as an application user within Business Central, mapping it to the BC2Fabric permission set.

3. Azure Key Vault configuration

  • Provision an Azure Key Vault in the same tenant that hosts Business Central and Microsoft Fabric.
  • Store the BC2Fabric client secret as individual secrets.
  • Configure access policies so that the BC2Fabric managed identity and operations team can retrieve secrets while limiting write access to administrators.

4. Microsoft Fabric capacity and workspace

  • Ensure your tenant has a Fabric capacity with available resources to run the BC2Fabric pipelines and lakehouse refreshes.
  • Create or select a dedicated workspace that will host the BC2Fabric assets.
