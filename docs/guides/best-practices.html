Operational Best Practices

Recommendations to keep your BC2 Fabric deployments stable, secure, and observable.

Security

  • Rotate API credentials every 90 days using the console or API.
  • Use scoped service accounts for automation tasks.
  • Enable IP allowlists for administrative endpoints.

Performance

  • Batch events to maximize throughput while respecting latency targets.
  • Leverage parallel pipelines to isolate heavy workloads.
  • Monitor queue depth and configure alerts for sustained spikes.

Observability

  • Forward logs to your SIEM using the webhook delivery option.
  • Create dashboards for key metrics such as success rate and retry volume.
  • Enable anomaly detection on high-value routes.
