Integration Guide

Connect BC2 Fabric to your data sources and destinations with a repeatable workflow.

1. Create a connector

In the console, create a new connector and select the desired template. Provide connection credentials and scope access to the relevant datasets.

2. Configure event mappings

Use schema inference or upload your own mapping rules. The platform validates transformations in real time to prevent runtime failures.

3. Activate routing rules

Define routing rules that reference your connector and specify retry policies. You can manage these rules via the UI or the REST API.

4. Monitor delivery

Track message delivery in the Observability tab. Combine live metrics with structured logs for end-to-end visibility.
