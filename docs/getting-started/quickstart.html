Getting Started

Quickstart

Follow these essentials to connect Microsoft Dynamics 365 Business Central with Microsoft Fabric using the BC2Fabric solution.

1. Install the Business Central extension

Start by installing the BC2Fabric extension from Microsoft AppSource in your Dynamics 365 Business Central environment. The extension enables the data synchronization services required for BC2Fabric.

Install BC2Fabric from AppSource

2. Secure application credentials

Register an application in Microsoft Entra ID and store the client ID and secret in an Azure Key Vault. The BC2Fabric Installation Guide walks through the required permissions, Key Vault configuration, and the connection details BC2Fabric needs.

3. Provision the Fabric workload

Install the BC2Fabric workload directly in Microsoft Fabric to create the workspace, pipelines, and datasets that power your analytics. Use the workload hub link below and sign in with the same tenant used for the Business Central extension.

Open the Microsoft Fabric workload hub

Once the workload is deployed, follow the BC2Fabric overview to connect your Business Central data sources and configure refresh schedules.
