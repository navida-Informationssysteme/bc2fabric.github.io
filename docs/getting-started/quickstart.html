Getting Started

Quickstart

Set up your environment and connect to BC2 Fabric in under 10 minutes.

Prerequisites

  • BC2 Fabric account with API access enabled.
  • Node.js 18+ or Python 3.11+ installed locally.
  • Access to your organization's staging environment.

1. Install the CLI

npm install -g @bc2fabric/cli

The CLI bootstraps projects, manages configuration, and inspects deployments.

2. Authenticate

bc2 login --client-id <CLIENT_ID> --client-secret <CLIENT_SECRET>

Credentials can be generated in the BC2 Fabric console. Authentication stores a short-lived session on your workstation.

3. Create a workspace

bc2 init --workspace my-first-app

The command scaffolds a minimal project, installs dependencies, and provisions a sandbox tenant for experimentation.

4. Deploy

bc2 deploy

Deployments automatically sync your configuration and emit structured logs you can inspect in the console or via webhooks.
