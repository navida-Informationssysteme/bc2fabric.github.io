Getting Started

BC2Fabric

BC2Fabric delivers a managed connection between Microsoft Dynamics 365 Business Central and Microsoft Fabric. It makes operational data available for analytics, reporting, and automation without custom integration projects.

What BC2Fabric provides

  • Ready-to-use data pipelines. Automated processes move Business Central entities into Microsoft Fabric Lakehouse and Warehouse storage.
  • Business-ready semantic models. Prebuilt Power BI datasets and reports accelerate insights for finance, sales, purchasing, and inventory teams.
  • Flexible extensibility. Include custom tables and fields from Business Central to tailor analytics to your organization.
  • Secure operations. Azure Key Vault integration keeps application credentials protected while the solution runs inside your Microsoft tenant.

How it works

  1. Install the BC2Fabric extension in your Business Central environment.
  2. Authorize the solution with Microsoft Entra ID and store secrets in Azure Key Vault.
  3. Deploy the managed workload in Microsoft Fabric to create pipelines and reports.
  4. Monitor refreshes and enrich the delivered datasets with your own dataflows or Lakehouse tables.

With these pieces in place, BC2Fabric keeps your Fabric workspace synchronized with operational data so you can build advanced analytics, automate processes, and share insights across the business.
